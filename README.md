
#  Electronic Gadgets Sales Dashboard

An interactive Excel sales performance dashboard designed to analyze revenue trends, regional performance, and customer engagement for an electronic gadgets retailer. 

This project demonstrates data cleaning, data modeling (Pivot Tables), and dynamic KPI reporting using Microsoft Excel.

###  Key Performance Indicators (KPIs) Captured
* **Total Sales Price:** ₹3,47,59,688
* **Total Units Sold:** 3,762 units
* **Total Products Handled:** 5 core electronic items
* **Total Customers Served:** 700 unique customers

---

###  Data Insights & Metrics
* **Top Selling Product:** The **Dell Inspiron 15** leads total revenue generation at over ₹1.28 Crore, followed closely by the **Mi NoteBook Ultra**.
* **Regional Dominance:** The **West** region accounts for the largest market share at **36%**, while the **East** region stands strong at **31%**. 
* **State/Territory Leader:** **Gujarat** generated the highest overall revenue (₹1,25,80,651) and logged the highest volume of units sold (1,315 units).
* **Monthly Trends:** Sales peaked significantly in **May** reaching ₹83,93,328, correlating directly with the highest volume of orders (147 orders) during that month.

---

###  Features & Tools Used
* **Data Cleaning & Wrangling:** Handled raw dataset formatting, currency localization (INR), and date structuring in Excel.
* **Analysis:** Pivot Tables and Pivot Charts for multi-dimensional data aggregation.
* **Interactivity:** Integrated dynamic **Slicers** for filtering data instantly by *Product Name* and *Order Date (Months/Years)*.
* **Design & Layout:** Styled with a clean, modern color palette, custom KPI cards, and custom data labels for optimal readability.

---

### 📂 How to View the Project
1. Clone this repository or download the `.xlsx` file directly.
2. Open the file in Microsoft Excel (ensure macros/content are enabled if prompted).
3. Use the **Product Name** and **Order Date** slicers on the right side of the dashboard to filter the metrics dynamically.
